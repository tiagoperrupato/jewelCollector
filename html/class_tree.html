Detailed Description

Classe que representa um obstáculo do tipo árvore. Ela herda a classe Obstacle e implementa a interface IRecharge A árvore impede que o robô ultrapasse a posição ocupada por ela, servindo como barreira. Além disso, pode servir para recarregar 3 unidades de energia do robô.
